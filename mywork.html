Week 1: Research

Identifying The Gap

I began by analyzing existing college websites like Banasthali, RIET, and RCEW.

  • Outdated aesthetics
  • Cluttered navigation
  • Lack of interactivity

The Shortfall

Most academic platforms felt static and uninspiring. They served as mere digital brochures rather than engaging platforms. This presented a massive opportunity for innovation.

Our Strategy

The Vision

To build a platform that completely fills this gap. I envisioned a highly interactive, premium digital ecosystem that feels alive and instantly wows the user, setting a completely new benchmark for academic and program websites.

The Design Language

Glassmorphism

I abandoned flat design in favor of "Glassmorphism". By using frosted glass UI elements over vibrant blur orbs and deep dark backgrounds, I created a sense of depth and a state-of-the-art aesthetic.

Deep Dark Theme

Provides high contrast for vibrant content.

Fluid Gradients

Living colors that catch the user's eye.

Challenge #1

Scalable Navigation

The Problem: The main index page was initially acting like a Single-Page Application (SPA) using Javascript modals. As content expanded, the modal system became cluttered, restrictive, and terrible for SEO.

The Solution

I re-engineered the routing architecture. I stripped out the complex JavaScript modal logic and replaced it with direct multi-page URL routing. This ensured absolute scalability and improved SEO instantly.

The Solution

I isolated the bug by identifying conflicts between CSS systems. To fix it globally without disrupting bespoke styles, I swapped conflicting utility classes for precise, pure inline CSS constraints.

Challenge #2

The "Giant Logo" Bug

The Problem: When mixing technologies (Tailwind CSS and Vanilla CSS) across different pages, layout breaks occurred. Most notably, the main AIC-JKLU logo blew up to cover the entire screen on specific program pages.

Challenge #3

Creating Consistency

Generating new pages like rise.html required perfectly matching the ultra-modern "Glassmorphism" look of the landing page. Building it poorly would lead to a disjointed brand identity.

The Solution

I meticulously abstracted the core CSS animations and Tailwind utility variables from the main codebase. This allowed me to easily port complex UI concepts—animated backgrounds, frosted glass layers, and blurred navigation—into fresh documents seamlessly.
